# PIKURR — Документация

## Система мониторинга агрополей

PIKURR — система автоматической оценки состояния агрополей по спутниковым данным.

### Контуры системы

| Контур | Папка | Назначение |
|---|---|---|
| Внутренний (ETL) | `PIKURR/` | ML-обработка, сборка пакетов данных |
| Внешний (Frontend) | `REPIKURR/` | Веб-карта, GeoServer, доставка данных |

### Документация

- [Развёртывание ETL (PIKURR)](deploy-etl.md) — настройка рабочей машины администратора
- [Развёртывание фронтенда (REPIKURR)](deploy-frontend.md) — настройка сервера 192.168.251.190

### Быстрый старт

**Обновить данные и отправить на сервер:**
```bash
./package_and_push.sh
```

**Поднять фронтенд с нуля:**
```bash
cd ~/repikurr && bash deploy.sh
```
